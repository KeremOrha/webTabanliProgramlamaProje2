# Web Tabanlı Programlama Proje 2


Proje bir botanik bahçesinin koleksiyon, sergi ve etkinliklerinin görüldüğü bir web sitesidir.
**Üyeler** sergileri, etkinlikleri ve koleksiyondaki bitkileri ekleyebilir, silebilir ve değiştirebilir.
**Yöneticiler** ise bunlarla birlikte üye ekleyebilir, silebilir ve değiştirebilir.
